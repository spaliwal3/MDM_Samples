A course registration plugin for Georgia Trauma Foundation

== Description ==

This plugin is developed specifically for Georgia Trauma Foundation. By activating the plugin, A 'Courses' menu will show on the side menu. User should be able to add courses and view student registration status.

== Installation ==

1. Upload `course_registration.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to create a Course List page =

Add a new page and make sure its slug is 'course-list'. All published courses will be shown on this page.
